# JofiStudio

**JofiStudio** is a modern, multi-page **frontend web application** built with **Next.js 14**, focused on **pixel-perfect implementation**, rich animations, and full responsiveness across all screen sizes.

The project was developed based on a **provided design specification**, with precise attention to layout, spacing, typography, and animations to ensure **design-to-code accuracy**.

## Technical Overview

JofiStudio is implemented as a **multi-page Next.js application** using **React 18**, optimized for **static export (SSG)** while retaining flexibility for future server-side extensions.

The codebase follows a component-driven architecture and integrates multiple UI and animation libraries to deliver smooth, high-quality user interactions.

Key technical characteristics:
- Fully responsive layout (mobile, tablet, desktop, wide screens)
- Multi-page architecture with file-based routing
- Pixel-perfect UI implementation based on design assets
- Advanced animations and transitions
- Internationalization (i18n) support
- Static Site Generation (SSG)
